### 第一个脚本
###### 自己用到的一个xcode工程的编译、打包脚本
如果别人想要拿去使用，需要修改很多“写死的”地方，比如一些文件名，所以这个脚本可能并没有那么“智能”。但是，搞的太“智能”，就需要把脚本很多配置放在参数中，搞的像一个复杂的命令；而脚本是用来方便使用者自己一个人的，实在没有必要这么搞，动手改一下脚本就一劳永逸了，所以，这个脚本中有很多写死的部分。
